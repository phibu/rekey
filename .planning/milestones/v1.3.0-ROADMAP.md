## Key Decisions

- **Parallel phase execution** — phases 02 and 03 ran concurrently under one milestone since they're independent.
- **`ShowAdPasswordPolicy` defaults to false** — opt-in so operators with restricted RootDSE access see no change.
- **HIBP response shape `{ suffixes, unavailable }`** (not `{ breached, count }`) — strict k-anonymity; the server never learns which suffix matched. Client matches the full-hash suffix locally.
- **Clipboard readback guard** — clipboard is only cleared if its content still equals the generated password (protects user's unrelated paste).
- **PhysicalFileProvider for `/brand/*`** mounts with `ServeUnknownFileTypes = false` — non-image files in the brand directory are not exposed.
- **Installer provisioning** — brand directory created on every installer run with `Test-Path` guard (safe fresh install + upgrade).
